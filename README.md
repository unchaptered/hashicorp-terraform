├ basic         # Basic Syntax

## Get Started

`terraform init`을 사용하면 다음의 경로에 Infra Provider와 소통하기 위한 API가 다운로드 됩니다. 

```cmd
~\.terraform\providers\registry.terraform.io\hashicorp\aws\4.57.1\windows_386\terraform-provider-aws_v4.57.1_x5.exe
```
